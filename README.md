# LVS Interview Task

This is a Flutter interview task developed using the **MVVM architecture** with **Provider** for state management.

## Features
- Google Sign-In with Firebase Authentication
- MVVM Architecture
- Provider State Management
- Reusable Widgets
- Clean Folder Structure

## Project Structure
- `helper` – App constants and styles
- `common_widgets` – Reusable UI components
- `model` – Data models
- `provider` – Business logic and state management
- `screen` – UI screens
- `service` – Firebase Google Authentication

## Tech Stack
- Flutter
- Dart
- Firebase Authentication
- Provider

## Getting Started
1. Clone the repository.
2. Run `flutter pub get`.
3. Configure Firebase for your platform.
4. Run the application.
